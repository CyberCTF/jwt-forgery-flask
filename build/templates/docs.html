üß™ Testing Examples

        
Use these curl commands to test the API endpoints:

        
        

            

                
Login and get token:

                
curl -X POST http://localhost:3206/api/login \
  -H "Content-Type: application/json" \
  -d '{"username": "john.doe", "password": "Welcome2024!"}'

            

            
            

                
Get user profile (with token):

                
curl -X GET http://localhost:3206/api/user/profile \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

            

            
            

                
Access admin endpoint:

                
curl -X GET http://localhost:3206/api/admin/system-config \
  -H "Authorization: Bearer ADMIN_JWT_TOKEN"
